### 8.Toast(提示)
* 接口:提示信息+时间
* 示例:
```js
toast.init('Hello World',500)
```

### 9.navFixed(导航栏固定)
* 接口:导航栏wrapper
* 示例:
```js
fixed.init($('.navWrapper'))
```

### 10.backToTop(回到顶部)
* 接口:回到顶部的时间,出现位置
* 示例:
```js
gotop.init(300,500)
```
